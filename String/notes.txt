# Java String & Character Methods - Quick Reference

## Character Class Methods

Character.isUpperCase(ch)  // Check if char is uppercase
Character.isLowerCase(ch)  // Check if char is lowercase
Character.isDigit(ch)      // Check if char is digit
Character.isLetter(ch)     // Check if char is letter


## String Case Methods

str.toUpperCase()          // Convert to uppercase
str.toLowerCase()          // Convert to lowercase
str.equals(str.toUpperCase())  // Check if string is all uppercase
str.equals(str.toLowerCase())  // Check if string is all lowercase


## Important Notes
- **String class has NO** `isUpperCase()` or `isLowerCase()` methods
- Use `Character` class for single character operations
- For strings, use `equals()` with `toUpperCase()/toLowerCase()` for case checking
- String concatenation in loops is inefficient - consider `StringBuilder`
